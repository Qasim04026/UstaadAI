# UstaadAI - اردو بیالوجی استاد

An offline-capable Urdu Biology tutor for Pakistani Class 9 FBISE students using Google Gemma 4.

## Demo
🤗 [Live Demo on HuggingFace Spaces](https://huggingface.co/spaces/qasim-robotic/ustaadai-demo)

## Features
- Pure Urdu responses
- FBISE Class 9 Biology syllabus
- MCQ with answers
- Pakistani context examples

## Model
- Base: Google Gemma 4 E2B
- Fine-tuned on 335 Urdu Biology Q&A dataset

## Dataset
- 335 examples covering all Class 9 Biology topics
- Available on Kaggle: qasim26/ustaadai-biology-dataset

## Tech Stack
- Google Gemma 4 E2B
- HuggingFace Transformers
- Gradio
- Kaggle T4 GPU
